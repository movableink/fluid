# Fluid Banner

This Ember component is a shortcut for these styles inherited from the [@movable/tailwind-config - banner component](https://movableink.github.io/tailwind-config/?path=/story/components-banners--all-types)

**Sections**
- [Destructive](#destructive)
- [Informative](#informative)
- [Confirmation](#confirmation)
- [Alert](#alert)
- [Default](#default)
- [Icon](#icon)
- [Closing](#closing)
- [Text Only](#text-only)

## Destructive

<Story id="components-fluid-banner--destructive" name="Fluid Banner" height="150px"/>


```hbs
<FluidBanner
  @header="Destructive Banner"
  @type="destructive"
  @icon="fluid-banner-destructive"
  @onClose={{this.mockAction}}
>
  <p>
    Body Content
  </p>
</FluidBanner>
```

## Icon

When passing the icon argument to the component use the full filename so `svg-jar` can render the icon property

<Story id="components-fluid-banner--without-close" name="Fluid Banner" height="150px"/>


```hbs
<FluidBanner
  @header="Default Banner"
  @icon="fluid-banner-archive"
>
  <p>
    Body Content
  </p>
</FluidBanner>
```

## Closing

The `close` icon will automatically appear when passing an `onClose` argument to the component.

<Story id="components-fluid-banner--without-icon" name="Fluid Banner" height="150px"/>


```hbs
<FluidBanner
  @header="Default Banner"
  @onClose={{this.mockAction}}
>
  <p>
    Body Content
  </p>
</FluidBanner>
```
## Text Only

If you don't define `@type` or `@onClose` the component with render as text only.

<Story id="components-fluid-banner--text-only" name="Fluid Banner" height="150px"/>


```hbs
<FluidBanner
  @header="Default Banner"
>
  <p>
    Body Content
  </p>
</FluidBanner>
```

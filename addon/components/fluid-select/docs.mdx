# Fluid Select

This is our select dropdown based off of [Ember Basic Dropdown](https://ember-basic-dropdown.com)

## Arguments Table

<details>
  <dl>
    <dt>options</dt>
    <dd>
      type: array
      required: true
      description: array of options to be displayed
    </dd>
    <dt>label</dt>
    <dd>
      <ul>
        <li>type: string</li>
        <li>required: false</li>
        <li>description: label associated with the select</li>
      </ul>
    </dd>
    <dt>select</dt>
    <dd>
      <ul>
        <li>type: function</li>
        <li>required: true</li>
        <li>description: what happens when the select is changed</li>
      </ul>
    </dd>
    <dt>selected</dt>
    <dd>
      <ul>
        <li>type: string | value</li>
        <li>required: false</li>
        <li>description: what value is active within the select</li>
      </ul>
    </dd>
    <dt>disabled</dt>
    <dd>
      <ul>
        <li>type: boolean</li>
        <li>required: false</li>
        <li>description: disables select</li>
      </ul>
    </dd>
    <dt>labelPath</dt>
    <dd>
      <ul>
        <li>type: string</li>
        <li>required: false</li>
        <li>description: when passing option object to select option</li>
      </ul>
    </dd>
    <dt>multiple</dt>
    <dd>
      <ul>
        <li>type: boolean</li>
        <li>required: false</li>
        <li>description: enabled multiple selection of options</li>
      </ul>
    </dd>
    <dt>search</dt>
    <dd>
      <ul>
        <li>type: function</li>
        <li>required: false</li>
        <li>description: async ability to query on user input</li>
      </ul>
    </dd>
    <dt>equalityFn</dt>
    <dd>
      <ul>
        <li>type: function</li>
        <li>required: false</li>
        <li>description: replace object equality with a custom function</li>
      </ul>
    </dd>
  </dl>
</details>

## Basic

Takes an array of options and renders it to the screen.

<Story
  id="components-fluid-select--default"
  height="200px"
/>

```hbs
<FluidSelect @options={{array 'apple' 'banana' 'orange'}} @label='fruit' />
```

## Grouped options
Takes a grouped array of options:

```js
const groupedOptions = [
  { groupLabel: 'Group one', groupOptions: ['one', 'two', 'three'] },
  { groupOptions: ['four', 'five', 'six'] },
  { groupLabel: 'Group three', groupOptions: ['seven', 'eight', 'nine', 'ten'] },
]
```

<Story
  id="components-fluid-select--grouped-options"
  height="200px"
/>

```hbs
<FluidSelect @options={{this.groupedOptions}} @label='Grouped Options' />
```

## Multiple

Can select multiple options

<Story
  id="components-fluid-select--multiple"
  height="200px"
/>

```hbs
<FluidSelect @options={{options}} @selected={{selected}} @select={{select}} @multiple={{true}} />
```

## Searching

<Story id="components-fluid-select--search" height="300px"/>

Used to search in listed items.

```hbs
<FluidSelect @options={{options}} @search={{search}} @select={{action (mut selected)}} />
```

## Block Mode

<Story id="components-fluid-select--block" height="180px"/>

```hbs
<FluidSelect @select={{action selectOption}} @selected={{selected}} as |select|>
  <select.trigger @label='Click Me!' />

  <select.popup>
    <select.list>
      {{#each options as |option|}}
        <select.option @option={{option}} />
      {{/each}}
    </select.list>
  </select.popup>
</FluidSelect>
```

# Fluid Modal

Our modal uses [Ember HeadlessUI's Dialog](https://gavinjoyce.github.io/ember-headlessui/dialog/dialog-modal) as a base.

The modal closes by default by hitting the `esc` key or by `clicking` outside the modal.
